🎁 Amigo Secreto / Secret Santa





Aplicação web simples e interativa para organizar e sortear nomes em um jogo de Amigo Secreto.
O usuário adiciona nomes, visualiza a lista e pode realizar sorteios individuais, removendo automaticamente os nomes já sorteados.


📌 Funcionalidades
➕ Adicionar nomes à lista.

🚫 Evitar nomes duplicados.

📜 Listar todos os nomes adicionados.

🎯 Sortear um amigo secreto aleatoriamente.

🔄 Reiniciar a lista e o sorteio.

🛠 Tecnologias Utilizadas
HTML5 – Estrutura da aplicação.

CSS3 – Estilização e responsividade.

JavaScript – Lógica de adição, sorteio e reinício.

🌍 English Version
A simple and interactive web app to organize and draw names for a Secret Santa game.
Users can add names, see the list, and draw random names while automatically removing them from the list.
